# Academic Literature Search Templates

## Search Strategy
For each hybrid algorithm proposal from Evo, conduct comprehensive literature search using these templates.

## Template 1: Basic Hybrid Search
```
"[Algorithm 1] [Algorithm 2] traveling salesman problem"
"hybrid [Algorithm 1] [Algorithm 2] TSP"
"combination of [Algorithm 1] and [Algorithm 2] for TSP"
"[Algorithm 1] combined with [Algorithm 2] traveling salesman"
```

## Template 2: Metaheuristic Combination Search
```
"metaheuristic hybridization traveling salesman"
"hybrid metaheuristic TSP [year]"
"combined heuristic approach traveling salesman"
"multi-method algorithm TSP"
```

## Template 3: Performance Comparison Search
```
"[Algorithm name] benchmark TSP 500 nodes"
"TSP heuristic comparison 500 cities"
"traveling salesman problem large instance performance"
```

## Template 4: Novelty Verification Search
```
"novel hybrid algorithm traveling salesman"
"new combination TSP heuristic"
"original approach traveling salesman problem"
```

## Search Databases Priority Order
1. **Google Scholar** - Broad coverage
2. **IEEE Xplore** - Engineering/CS focus
3. **ACM Digital Library** - Computer science
4. **arXiv** - Preprints and recent work
5. **ScienceDirect** - Journal articles
6. **SpringerLink** - Conference proceedings

## Search Timeframe
- Primary: Last 10 years (2016-2026)
- Secondary: Historical (pre-2016) for foundational work
- Always check for recent preprints on arXiv

## Documentation Requirements
For each search:
1. **Search Query**: Exact query used
2. **Database**: Where search was conducted
3. **Date**: When search was performed
4. **Results**: Number of results, key papers found
5. **Relevance Assessment**: How relevant to proposed hybrid
6. **Novelty Conclusion**: Whether approach exists in literature

## Quality Assessment Criteria
- **Conference/Journal Prestige**: Top venues (NeurIPS, ICML, AAAI, etc.)
- **Citation Count**: Highly cited papers indicate established work
- **Publication Date**: Recent work more relevant for novelty
- **Methodological Similarity**: How similar to proposed approach
- **Performance Claims**: Comparable benchmark results

## Red Flags for Non-Novelty
1. Multiple papers describing similar hybrid
2. Established name for the hybrid approach
3. Survey papers listing it as known method
4. Textbook coverage of the approach
5. Software implementations available
